📊 技术分析实战

今日学习：MA 移动平均线 + MACD

今天系统学习了两个最常用的技术指标，用实战案例加深理解：

MA（移动平均线）

定义：把过去 N 天的价格平均一下，画成一条线

指标 全称 含义
MA5 5日均线 短期成本，最近一周的平均价格
MA20 20日均线 中期成本，最近一个月的平均价格
MA60 60日均线 长期成本，最近三个月的平均价格

实战应用：

  • MA5 > MA20：短期走势 > 中期走势 → 偏多 🟢
  • MA5 < MA20：短期走势 < 中期走势 → 偏空 🔴
  • 金叉：MA5 上穿 MA20 → 买入信号 🟢
  • 死叉：MA5 下穿 MA20 → 卖出信号 🔴

MACD（指数平滑异同移动平均线）

定义：衡量短期和长期趋势的速度差异

指标 含义
DIF（快线） 短期和长期的 EMA 差值，反映变化速度
DEA（慢线） DIF 的平滑线
MACD 柱 DIF - DEA，反映加速/减速

实战应用：

  • MACD > 0：短期涨得比长期快 → 多头 🟢
  • MACD < 0：短期跌得比长期快 → 空头 🔴
  • MACD 由负转正：动能转强，可能启动上涨
  • MACD 由正转负：动能转弱，可能开始下跌
